## 为什么写这个系列

n8n 是一个极具工程价值的自动化平台，但在实际使用中，**90% 的问题并不出在工作流本身，而是出在部署、运行与维护方式上**。

这个系列记录了一套**从零到长期稳定运行**的实践路径，目标不是“跑起来”，而是：

- 可维护  
- 可升级  
- 可迁移  
- 可长期依赖  

## 内容结构

- **新手使用手册**
  - 面向第一次接触 n8n 的使用者
  - 重点解决“怎么用、不踩坑”

- **维护运维手册**
  - 面向已经开始长期使用 n8n 的场景
  - 重点解决“怎么稳、不出事故”

## 适用环境说明

- Docker Desktop（Windows / macOS）
- docker compose（v2）
- n8n 官方镜像
- 单机、本地运行场景

后续如涉及公网部署、反向代理、Postgres，将在此专题中扩展。
